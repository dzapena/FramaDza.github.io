Installations

Untuk dapat menggunakan FramaDza Framawork, kamu harus memasang frameworknya terlebih dahulu. Ada dua cara instalasi FramaDza.

  • Offline Installation
  • Online CDN

Offline Installation

Cara ini adalah cara tercepat dan paling fleksibel. Karena selain mudah diedit, anda juga tidak memerlukan internet untuk dapat mengaksesnya. Berikut cara instalasinya

  1. Unduh FramaDza melalui link diatas.
  2. Ekstrak ke lokasi dimana anda menyimpan project website kalian
  3. dalam file index.html anda, tambahkan referensi berikut pada bagian header website anda <link rel="stylesheet" href="fdz/css/main_framadza.css">

Online Installation

Cara ini adalah cara termudah, tapi kamu harus selalu online untuk memastikan website kamu terhubung dengan FramaDza. Berikut cara instalasinya

  1. Salin kode ini <link rel="stylesheet" href="cdn.frama.dza.my.id/fdz/css/main_framadza.css">
  2. Tempelkan pada bagian header website kalian

Components Installation

Dengan Components, kamu akan mendapatkan kemudahan dan menghemat banyak waktu untuk membuat atau mengembangkan webiste kalian. Berikut cara instalasinya

  1. Cukup mudah, yang perlu dilakukan hanya Salin kode yang terdapat pada FramaDza terkait components apa yang ingin anda pasang
  2. Selesai, kamu mendapatkan seluruh deisgnya begitu saja
