What is FramaDza ?

FramaDza adalah Framework CSS sederhana yang dibuat hanya untuk keperluan pribadi sebagai bahan ajar dan eksperimen. FramaDza terdiri dari Class Class CSS sederhana, dibantu oleh Font Awesome dalam menambahkan icon. Project ini bisa anda gunakan gratis, selama ada credits yang menyertakan FramaDza pada bagian Footer website anda yang dibangun menggunakan Framework ini.
Terima kasih.

Why FramaDza ?

Framadza membantu anda membuat website dengan lebih cepat dan mudah. Yang anda perlukan hanya memasukan class. Selain itu. FramaDza sangat cocok untuk membuat website sederhana seperti Portofolio, Website Profile Perusahaan, Blog dan Sejenisnya.
